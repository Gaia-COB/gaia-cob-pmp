## Pre-requisites

You need Python >=3.9 on your machine to install `iommi-demo`.

## Install with `pip`

```bash
pip install iommi-demo
```

## Install with `conda`

```bash
conda install iommi-demo --channel conda-forge
```

## Source installation

For other platforms and for development installations, [build `iommi-demo` from source](Build-from-Source).
